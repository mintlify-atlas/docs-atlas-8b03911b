<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with Happy in minutes</p>
  
  <Steps>
    <Step title="Download the mobile app">
      Install Happy on your iOS or Android device to control your AI agents remotely.
      
      - [Download on the App Store](https://apps.apple.com/us/app/happy-claude-code-client/id6748571505)
      - [Download on Google Play](https://play.google.com/store/apps/details?id=com.ex3ndr.happy)
      - [Open Web App](https://app.happy.engineering)
    </Step>
    
    <Step title="Install the CLI">
      Install the Happy CLI on your computer to wrap Claude Code and Codex commands.
      
      <CodeGroup>
      ```bash npm
      npm install -g happy-coder
      ```
      
      ```bash yarn
      yarn global add happy-coder
      ```
      
      ```bash pnpm
      pnpm add -g happy-coder
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Authenticate with QR code">
      Run the `happy` command and scan the QR code with your mobile app to link your devices.
      
      ```bash
      happy
      ```
      
      <Note>
        Your authentication uses end-to-end encryption. The QR code contains a one-time challenge that establishes a secure connection between your devices.
      </Note>
    </Step>
    
    <Step title="Start coding with AI">
      Use `happy` instead of `claude` to start sessions with mobile control enabled.
      
      ```bash
      # Instead of: claude
      happy
      
      # Instead of: codex
      happy codex
      
      # Resume previous sessions
      happy --resume
      ```
      
      Now you can monitor and control your AI coding sessions from your phone, receive push notifications when input is needed, and switch control between devices instantly.
    </Step>
  </Steps>
</div>
